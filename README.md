## Food Ordering App

This is an Android-based Food Ordering Application developed using Kotlin in Android Studio with Firebase integration. The app allows users to browse different food items, view details, select food size, add items to a cart, and place orders through a simple and user-friendly interface.

The application also includes user authentication, profile management, and an order summary page where users can view the total bill and confirm their delivery address fetched from Firebase.

This project demonstrates the use of Android UI components, RecyclerView, Firebase Authentication, and Firebase Firestore for storing user data and managing orders.

## Features
- User Signup and Login using Firebase Authentication
- Browse multiple food items
- View food details and select size options
- Add items to cart with quantity
- View cart and proceed to checkout
- Order summary with total bill calculation
- Fetch delivery address from Firebase
- Place order confirmation screen

## Technologies Used
- Kotlin
- Android Studio
- Firebase Authentication
- Firebase Firestore
- RecyclerView
- XML UI Design

## Purpose
The purpose of this project is to build a simple food ordering system and gain practical experience in Android development, Firebase integration, and mobile app UI design.
